About the Project

WDU Trust (WDU) is a transparent and fixed-supply BEP-20 utility token deployed on the BNB Smart Chain.

The smart contract is fully verified and includes no minting, burning, transaction tax, blacklist, or hidden owner functions.

Project Principles

Transparency

The source code is publicly available and verified on BscScan, allowing independent review of the contract logic.

Fixed Supply

WDU Trust has a fixed total supply with no mint or burn mechanisms, ensuring predictable token behavior.

Utility-Oriented

The token is designed for basic Web3 utilities and educational use cases.

Team & Governance

This project does not operate with a publicly promoted team, venture capital, or private investors.

Maintenance and updates are handled transparently through on-chain verification and public documentation.

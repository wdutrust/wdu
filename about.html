About the Project

WDU Trust (WDU) is a fixed-supply ERC-20 utility token focused on transparency, simplicity, and verifiable on-chain behavior.

The smart contract is fully verified and contains no minting, burning, taxation, or restrictive administrative mechanisms.

Project Principles

Transparency

All smart contract source code and documentation are publicly available and verifiable via blockchain explorers and GitHub.

Predictability

A fixed total supply and absence of hidden functions ensure predictable token behavior over time.

Utility-Oriented

Designed for educational, experimental, and basic Web3 utility use cases.

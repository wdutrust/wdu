Project Overview

WDU Trust Token (WDU) is an independently maintained BEP-20 utility token deployed on the BNB Smart Chain. The project is designed for transparency, educational usage, and simple, verifiable blockchain interactions.

The smart contract is fully verified and immutable. It contains no minting, burning, transaction taxes, blacklist functions, or administrative control mechanisms.
